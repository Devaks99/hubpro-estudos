# 🚀 HubPro Estudos - Hub Protheus

Plataforma de estudos focada em **Protheus e ADVPL**, com sistema de flashcards, quizzes e acompanhamento de desempenho.

---

## 📌 Sobre o Projeto

O **HubPro Estudos** é uma aplicação web desenvolvida com:

- HTML5
- JavaScript
- Tailwind CSS

O objetivo é fornecer um ambiente moderno e intuitivo para aprendizado técnico, utilizando:

- 📚 Flashcards inteligentes
- 🧠 Sistema de Quiz
- 📊 Dashboard de desempenho
- 📈 Controle de progresso

---

## 🎯 Funcionalidades

✅ Interface moderna estilo SaaS  
✅ Navegação lateral fixa  
✅ Sistema de Flashcards  
✅ Sistema de Quiz com pontuação  
✅ Dashboard com métricas de desempenho  
✅ Layout responsivo
